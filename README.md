# CatBlog 🐱
A simple HTML blog page dedicated to **Mr. Whiskers**, a playful and lovable cat.

## Features
* Semantic HTML structure
* Navigation menu with anchor links
* About section
* Multiple blog posts
* Contact information
* Responsive viewport settings
* Figure and figcaption elements for images

## Preview
The page includes:
* A welcome header
* A photo of Mr. Whiskers
* An About section introducing the author and cat
* Three blog posts:
  * Mr. Whiskers' First Day Home
  * Mr. Whiskers' First Bath
  * Mr. Whiskers' First Birthday Party
* Contact details in the footer

## Technologies Used
* HTML5

## How to Run
1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/catBlog.git
   ```
2. Open the project folder.
3. Open `index.html` in your web browser.
